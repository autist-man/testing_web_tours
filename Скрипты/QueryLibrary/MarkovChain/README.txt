В файле "main.c" реализован алгоритм собирающий статистику для цепочки Маркова.
Результатом работы алгоритма является стахостическая матрица(матрица вероятностей), записанная в файл "stochastic_matrix.txt".
Данные для матрицы собираются из файла "real_usernames_datapool.txt". В файле должны быть только буквы английского алфавита в верхнем и нижнем регистре.
